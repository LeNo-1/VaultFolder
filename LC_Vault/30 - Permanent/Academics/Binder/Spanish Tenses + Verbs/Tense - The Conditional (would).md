## Irregular verbs in el condicional

The irregular verbs in el conditional tense are exactly the same as those in the future tense. The conditional endings will remain the same throughout but there will be changes to the stem or root just as in the future tense. The same 12 irregular verbs apply in the conditional tense:

### 1. Verbs that drop the e or i and add d:

| Infinitive | Future  | Conditional |
| :--------- | :------ | :---------- |
| poner      | pondré  | pondría     |
| salir      | saldré  | saldría     |
| tener      | tendré  | tendría     |
| valer      | valdré  | valdría     |
| venir      | vendré  | vendría     |

### 2. Verbs that drop the final vowel completely:

| Infinitive | Future  | Conditional |
| :--------- | :------ | :---------- |
| poder      | podré   | podría      |
| caber      | cabré   | cabría      |
| haber      | habré   | habría      |
| querer     | querré  | querría     |
| saber      | sabré   | sabría      |

### 3. Decir and hacer are both special cases with irregular stems:

| Infinitive | Future | Conditional |
| :--------- | :----- | :---------- |
| decir      | diré   | diría       |
| hacer      | haré   | haría       |

Compounds of these verbs will also be irregular following the same structure:

| Infinitive |       | Future      | Conditional |
| :--------- | :---- | :---------- | :---------- |
| componer   | to fix | compondré   | compondría  |
| disponer   | to dispose | dispondré   | dispondría  |
| imponer    | to impose | impondré    | impondría   |
| proponer   | to propose | propondré   | propondría  |
| suponer    | to suppose | supondré    | supondría   |
| reponer    | to replace | repondré    | repondría   |
| detener    | to detain | detendré    | detendría   |
| contener   | to contain | contendré   | contendría  |
| mantener   | to maintain | mantendré   | mantendría  |
| obtener    | to obtain | obtendré    | obtendría   |
| retener    | to retain | retendré    | retendría   |
| deshacer   | to undo | desharé     | desharía    |
| rehacer    | to do again | reharé      | reharía     |
