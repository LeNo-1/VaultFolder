## Gustar

Indirect object pronouns

| me  | Me gusta  | Me gustan  |
| --- | --------- | ---------- |
| te  | te gusta  | te gustan  |
| le  | le gusta  | le gustan  |
| le  | le gusta  | le gustan  |
| nos | nos gusta | nos gustan |
| os  | os gusta  | os gustan  |
| les | les gusta | les gustan |
For clarity and emphasis Spanish can place a pronoun with the preposition 'a' before the indirect object:

| A mí          |     |
| ------------- | --- |
| A ti          |     |
| A él          |     |
| A ella        |     |
| A usted       |     |
| A nosotros    |     |
| A vosotros    |     |
| A ellos/ellas |     |
| A ustedes     |     |
